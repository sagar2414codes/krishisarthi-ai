# 🌾 KrishiSarthi AI

KrishiSarthi AI is an AI-powered smart farming assistant designed to help farmers and agricultural planners make data-driven decisions using machine learning and modern web technologies.

---

## 🚀 Features

- 🌱 Crop Recommendation System
- 🦠 Disease Detection using AI
- 📊 Yield Prediction
- 💰 Profit Analysis
- 🌦 Weather-Based Insights
- 🧠 Smart Advisory System
- 📈 Interactive Dashboard
- 📱 Responsive UI

---

## 🧠 Problem Statement

Farmers often struggle with:
- selecting the right crop
- identifying diseases early
- estimating yield and profit

KrishiSarthi AI solves this by providing intelligent recommendations using AI models.

---

## 💡 Solution

This platform integrates:
- machine learning models
- real-time inputs
- user-friendly interface

to deliver actionable farming insights.

⚙️ Setup Instructions
1️ Clone the repository
git clone https://github.com/your-username/krishisarthi-ai.git
cd krishisarthi-ai

2️ Backend Setup
cd backend
py -3.11 -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py

2️⃣ Backend Setup
cd backend
py -3.11 -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py

If requirements.txt does not work:

pip install fastapi uvicorn flask pandas numpy scikit-learn pillow opencv-python tensorflow python-multipart pytho

3️ Frontend Setup
cd frontend
npm install
npm run dev

4️ Environment Variable
Create a file named .env inside the frontend folder and add:
VITE_API_BASE_URL=http://127.0.0.1:7860
